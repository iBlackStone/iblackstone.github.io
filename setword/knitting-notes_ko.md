# 뜨개 노트 도움말

뜨개 노트에서는 PDF 도안을 가져와 읽고, 페이지 위에 자, 텍스트, 이미지, 정보 핀, 브러시 선, 카운터를 추가할 수 있습니다. 페이지 요소를 추가하거나 그려도 원본 PDF 자체는 직접 변경되지 않습니다.

## 프로젝트 만들기와 열기

1. 도구 화면에서 **뜨개 노트**를 선택합니다.
2. 가져올 PDF 파일을 선택합니다.
3. 가져오기가 끝나면 뜨개 노트 프로젝트 목록에 표시됩니다.
4. 프로젝트를 탭하면 이전에 보던 위치와 편집 상태에서 계속할 수 있습니다.

무료 사용자는 뜨개 노트 프로젝트를 1개까지 만들 수 있습니다. 프로젝트 삭제는 되돌릴 수 없으므로 더 이상 필요하지 않은지 확인한 뒤 삭제해 주세요.

## 편집기 화면

### 상단 도구 모음

- **닫기**: 프로젝트 목록으로 돌아가며 현재 읽기 상태를 저장합니다.
- **실행 취소**: 지원되는 최근 브러시 또는 구성 요소 작업을 취소합니다.
- **다시 실행**: 방금 취소한 작업을 복원합니다.
- **사용 안내**: `?`를 탭하면 언제든 이 도움말을 열 수 있습니다.
- **더보기**: 현재 프로젝트의 이름을 바꾸거나 저장 또는 삭제합니다.

### 도구 모음 표시와 숨기기

PDF의 빈 영역을 탭하면 상단과 하단 도구 모음을 숨겨 읽기 영역을 넓힐 수 있습니다. 화면 가장자리의 작은 버튼을 탭하면 다시 표시됩니다.

### 기본 읽기 제스처

- 한 손가락으로 밀기: PDF를 스크롤합니다.
- 두 손가락으로 오므리기: 페이지를 확대하거나 축소합니다.
- 페이지 구성 요소 탭: 구성 요소를 선택하고 플로팅 도구 모음을 표시합니다.
- PDF 스크롤 또는 확대·축소: 현재 구성 요소 선택을 해제합니다.

| 구성 요소 | 이미지, 정보 핀, 도안 링크, 텍스트 또는 자 선택 |
| 이동 | PDF 읽기, 스크롤, 확대·축소 모드로 돌아가기 |

| 지우개 | 브러시로 그린 선 지우기 |

| 더보기 | 요소 관리, 기본 색상, 브러시 관리, 자 관리 열기 |

## 페이지와 라벨

하단의 **페이지**를 탭하면 미리보기 패널이 열립니다.

- 미리보기를 탭하면 해당 페이지로 이동합니다.
- 상단 라벨 목록에서 라벨이 지정된 페이지로 바로 이동할 수 있습니다.
- 페이지 라벨은 미리보기 아래쪽에 표시됩니다.
- 선택한 미리보기의 `...`를 탭해 라벨을 추가, 편집 또는 삭제할 수 있습니다.
- 페이지마다 라벨은 1개만 설정할 수 있습니다.

무료 사용자는 페이지 라벨을 2개까지 만들 수 있습니다. 기존 라벨은 계속 편집하거나 삭제할 수 있습니다.

## 페이지 구성 요소 추가

1. 하단의 **구성 요소**를 탭합니다.
2. 추가할 구성 요소를 선택합니다.
3. 구성 요소 버튼이 선택한 항목의 아이콘으로 바뀝니다.
4. PDF에서 배치할 위치를 탭합니다. 탭한 지점을 중심으로 구성 요소가 추가됩니다.
5. 추가가 끝나면 편집기가 자동으로 이동 모드로 돌아갑니다.

새 페이지 구성 요소 추가는 멤버십 기능입니다. 기존 구성 요소는 계속 확인하고 편집할 수 있습니다.

### 지원하는 구성 요소

#### 자

현재 읽고 있는 행이나 열을 따라갈 때 사용합니다. 기본으로 가로 자, 세로 자, 45도 자가 제공되며 자 관리에서 프리셋을 조정할 수 있습니다.

#### 텍스트

여러 줄 텍스트를 지원합니다. 텍스트 상자를 늘리면 더 많은 내용을 표시할 수 있습니다. 글자색, 글자 크기, 정렬, 복사, 붙여넣기를 설정할 수 있습니다.

#### 이미지

카메라, 파일 또는 사진 보관함에서 이미지를 선택할 수 있습니다. 원본 가로세로 비율로 추가되며 비율을 유지한 확대·축소와 회전을 지원합니다. 각 변을 따로 늘릴 수는 없습니다.

#### 정보 핀

페이지에 고정 크기의 `i` 아이콘으로 표시됩니다. 아이콘을 탭하면 내용을 확인할 수 있고, 팝업의 텍스트 영역을 두 번 탭하면 편집을 시작합니다.

#### 도안 링크

PDF를 앱 안의 기존 대바늘 도안과 연결합니다. 연결한 뒤에는 미리보기를 표시하고 PDF에서 해당 대바늘 도안으로 바로 이동할 수 있습니다.

#### 도안 격자 보정

도안 격자 보정은 PDF에 들어 있는 픽셀 도안이나 대바늘 도안을 행과 열 정보가 있는 재사용 가능한 격자 원본으로 바꿉니다.

1. **구성 요소**에서 **도안 격자**를 선택한 뒤 도안 영역을 탭해 외곽 프레임을 놓습니다.
2. 프레임 안의 빈 영역을 드래그해 위치를 맞추고 외곽 조작점으로 크기나 각도를 조정합니다.
3. 가로 가이드는 위쪽, 세로 가이드는 왼쪽에서 시작합니다. 각 가이드의 위치와 너비를 조정해 양쪽 가장자리를 이웃한 격자선 두 개에 맞춥니다.
4. **정렬**을 탭하면 두 축이 가장 가까운 완전한 주기에 맞춰집니다. 정확한 위치에 가까워지면 자동으로 스냅되고 햅틱 피드백이 제공됩니다.
5. 빨간색은 추가 조정이 필요한 상태입니다. 프레임과 격자를 확인하면 초록색 **정렬됨** 상태가 됩니다. 프레임을 이동하거나 크기 변경 또는 회전하면 다시 정렬해야 합니다.
6. **외곽 프레임 잠금**은 프레임만 고정하고 내부 가이드는 계속 조정할 수 있습니다. 뒤쪽의 **잠금**은 구성 요소 전체를 잠급니다.
7. **격자 이미지 미리보기**는 대바늘 작품을 만들지 않고 추출 결과를 확인합니다.
8. **카운터 도안 만들기**는 대바늘 도안 하나를 만들고 연결합니다. 이후에는 같은 항목으로 열거나 동기화하므로 매번 새 복사본을 만들지 않습니다.
9. **더보기**에서 보정 가이드/전체 격자를 전환하고 행과 열의 읽기 방향을 설정할 수 있습니다. 선택하지 않은 상태에서도 작은 격자 표시가 남으며 연결된 구성 요소는 두 번 탭해 바로 열 수 있습니다.

현재 버전은 사용자가 보정한 격자로 셀과 참고 이미지를 생성합니다. PDF 안의 뜨개 기호를 아직 자동으로 인식하지 않습니다. 사용 전에 미리보기에서 네 변, 행과 열 개수, 읽기 방향을 확인해 주세요.

## 페이지 구성 요소 편집

구성 요소를 탭하면 점선 선택 테두리, 조작점, 플로팅 도구 모음이 표시됩니다. 사용할 수 있는 작업은 구성 요소 종류에 따라 다릅니다.

- 구성 요소 안쪽 드래그: 위치를 이동합니다.
- 변의 조작점 드래그: 너비 또는 높이를 변경합니다.
- 오른쪽 위 조작점 드래그: 비율을 유지해 확대·축소합니다.
- 위쪽 조작점 드래그: 회전합니다.
- 색상: 구성 요소 색상을 변경합니다.
- 투명도: 배경 투명도를 조절합니다.
- 변형: 각도나 확대 비율을 빠르게 설정합니다.
- 삭제: 현재 구성 요소를 삭제합니다.

모든 구성 요소가 모든 작업을 지원하는 것은 아닙니다. 예를 들어 이미지는 각 변을 따로 늘릴 수 없고, 정보 핀과 도안 링크는 회전이나 확대·축소를 지원하지 않습니다.

### 잠금과 잠금 해제

- 잠그면 플로팅 도구 모음이 **잠금 해제** 버튼 하나로 줄어듭니다.
- 잠긴 자는 이동할 수 있지만 크기 변경, 확대·축소, 회전, 스타일 변경은 할 수 없습니다.
- 도안 격자의 **외곽 프레임 잠금**은 프레임의 실수 이동만 막고 내부 가이드는 계속 조정할 수 있습니다. 전체를 고정하려면 일반 **잠금**을 사용하세요.
- 잠긴 텍스트, 이미지, 정보 핀, 도안 링크 등은 이동하거나 편집할 수 없습니다.
- **잠금 해제**를 탭하면 전체 도구 모음과 조작점이 다시 표시됩니다.

## 브러시와 지우개

**브러시**에서 프리셋을 선택한 뒤 현재 페이지에 그릴 수 있습니다.

- 브러시 프리셋에는 모양, 굵기, 색상, 투명도가 포함됩니다.
- 브러시 관리에서 프리셋을 추가, 편집, 삭제, 정렬할 수 있습니다.
- 목록 위쪽의 브러시가 편집기에 우선 표시됩니다.
- 지우개는 브러시 선만 지웁니다. 자, 텍스트, 이미지 또는 다른 구성 요소는 삭제하지 않습니다.
- 구성 요소는 선택 후 삭제 버튼을 사용하거나 요소 관리에서 삭제해 주세요.

새 브러시 프리셋 추가는 멤버십 기능입니다. 기본 브러시와 기존 브러시는 계속 사용할 수 있습니다.

## 카운터

각 뜨개 노트 프로젝트에는 카운터가 최소 1개 포함됩니다.

- 숫자 탭: 1 증가합니다.
- `-` 탭: 1 감소합니다.
- 초기화 탭: 0으로 되돌립니다.
- `...` 탭 또는 카운터 길게 누르기: 카운터 메뉴를 엽니다.
- 메뉴에서 편집, 숫자 직접 입력, 색상 변경, 이름 변경, 삭제를 할 수 있습니다.
- 행 수는 카운터의 최댓값입니다. 최댓값을 넘으면 0부터 다시 셉니다.
- 프로젝트의 첫 번째 카운터는 삭제할 수 없습니다.

가로 한 페이지에는 카운터가 최대 3개 표시됩니다. 3개를 넘으면 좌우로 밀어 전환하며, 상단 페이지 표시기가 현재 위치를 알려 줍니다. 오른쪽 위 버튼으로 전체 높이와 간단 높이를 전환할 수 있습니다.

무료 사용자는 카운터를 3개까지 사용할 수 있습니다.

## 요소 관리

하단의 **더보기 > 요소 관리**에서 구성 요소와 브러시 메모를 한곳에서 관리할 수 있습니다.

- 왼쪽 미리보기로 요소의 내용과 대략적인 윤곽을 확인할 수 있습니다.
- 요소를 탭하면 해당 PDF 페이지와 위치로 이동합니다.
- 구성 요소를 잠그거나 잠금 해제할 수 있습니다.
- 필요 없는 요소를 삭제할 수 있습니다.
- 가까이 있는 브러시 선은 알아보기 쉬운 하나의 메모 미리보기로 묶여 표시됩니다.

## 기본 색상, 브러시와 자 프리셋

하단 **더보기**에는 공통 설정이 있습니다.

- **기본 색상**: 이후에 추가하는 페이지 구성 요소에 적용됩니다.
- **브러시 관리**: 모든 뜨개 노트 프로젝트가 함께 사용하는 브러시 프리셋을 관리합니다.
- **자 관리**: 모든 뜨개 노트 프로젝트가 함께 사용하는 자 프리셋을 관리합니다.

공통 프리셋을 변경해도 프로젝트에 이미 추가된 구성 요소나 선은 자동으로 바뀌지 않습니다.

## 저장, 실행 취소와 복원

- 읽던 페이지, 페이지 라벨, 카운터, 대부분의 편집 내용은 프로젝트와 함께 저장됩니다.
- 바로 저장하려면 상단의 **더보기 > 저장**을 사용합니다.
- 실행 취소와 다시 실행은 현재 편집 세션에서 지원되는 구성 요소와 브러시 작업에 적용됩니다.
- 프로젝트를 나갔다가 다시 열면 실행 취소 기록은 유지되지 않을 수 있지만, 저장된 페이지 내용은 복원됩니다.

## iCloud와 여러 기기

뜨개 노트 백업에는 프로젝트 데이터베이스, PDF 파일, PDF 관련 이미지 자료가 포함됩니다. 다른 기기에서 전체 프로젝트를 보려면 앱의 iCloud 동기화 또는 복원이 완료되어야 합니다.

주의 사항:

- iCloud 백업은 실시간 공동 편집 기능이 아닙니다.
- 큰 PDF는 업로드와 복원에 시간이 더 걸릴 수 있습니다.
- 복원이 완료되기 전에 원래 기기의 프로젝트나 PDF를 삭제하지 마세요.
- 데이터베이스만 포함한 이전 형식의 백업에는 원본 PDF가 없을 수 있습니다.

## 자주 묻는 질문

### 구성 요소를 선택한 뒤 PDF가 스크롤되지 않는 이유는 무엇인가요?

구성 요소의 터치 영역에서는 구성 요소 제스처가 PDF보다 우선합니다. 빈 영역을 탭하거나 이동 모드로 전환하거나 PDF 확대·축소를 시작하면 선택이 해제됩니다.

### 지우개로 자나 텍스트를 삭제할 수 없는 이유는 무엇인가요?

지우개는 브러시 선만 처리합니다. 다른 요소는 구성 요소의 삭제 버튼이나 요소 관리에서 삭제해 주세요.

### 잠긴 자가 계속 이동하는 이유는 무엇인가요?

도안을 따라가면서 자의 위치를 계속 조정해야 하기 때문입니다. 잠금은 자의 크기, 각도, 스타일을 고정하고 이동만 허용합니다. 다른 잠긴 구성 요소는 완전히 고정됩니다.

### 구성 요소를 선택한 뒤 페이지를 다시 탭하는 이유는 무엇인가요?

먼저 선택하고 배치하는 방식은 필요한 위치에 바로 추가할 수 있어 페이지 중앙에서 다시 옮기는 작업을 줄여 줍니다.

### 일부 기능에서 멤버십 안내가 표시되는 이유는 무엇인가요?

무료 버전은 기본 가져오기, 읽기와 제한된 사용량을 제공합니다. 페이지 구성 요소 추가나 브러시 프리셋 추가 같은 고급 기능에는 멤버십이 필요합니다.

<a id="getting-started"></a>
## 처음 시작하기

1. 홈에서 내장된 초보자 튜토리얼 작품을 열어 프로젝트, 카운터, 도안, 노트가 연결되는 방식을 확인합니다.
2. 사진, 규격, 재고, 가격이 포함된 실 기록을 만듭니다.
3. 프로젝트를 만들고 사용할 실을 연결합니다.
4. 카운터와 대바늘 도안, 코바늘 도안 또는 뜨개 노트를 추가합니다.
5. 다음에는 홈의 현재 프로젝트 카드에서 계속 작업합니다.

프로젝트가 하나도 없으면 내장 튜토리얼을 한 번 자동으로 가져옵니다. 이 과정에는 멤버십이 필요하지 않으며 무료 사용자의 개인 프로젝트 한도도 사용하지 않습니다. 직접 삭제하면 반복해서 다시 생성되지 않습니다.

<a id="main-navigation"></a>
## 주요 화면

- **홈**: 현재 프로젝트와 다음 작업을 표시하고 준비, 진행 중, 완료, 중단 등의 상태로 작품을 정리합니다.
- **실**: 실 기록 생성, 검색, 그룹, 재고, 달력을 관리합니다.
- **도구**: 도안 편집기, 카운터, 뜨개 노트, 색상, 기호, 계산 도구를 제공합니다.
- **설정**: 테마, 언어, 구독, 데이터 동기화, 업데이트 기록, 피드백, 개인정보 및 도움말을 관리합니다.

<a id="yarn-library"></a>
## 실 보관함

- 이름, 브랜드, 섬유, 색상, 형태, 합수, 번수, 무게, 길이, 가격, 재고를 기록할 수 있습니다.
- 여러 사진과 구매·사용 정보를 저장할 수 있습니다.
- 충분한 규격을 입력하면 선밀도, 굵기, 권장 바늘 크기, 길이 등을 계산합니다.
- 실을 프로젝트에 연결하고 실제 사용량을 기록할 수 있습니다.
- 복합 검색, 소재·색상 등의 그룹, 재고 경고, 실 달력을 이용할 수 있습니다.

### 스와치 시뮬레이터

실 상세 화면에서 엽니다. 코 수, 단 수, 완성 크기 또는 게이지를 입력해 밀도와 목표 크기를 환산합니다.

<a id="project-workspace"></a>
## 프로젝트 작업 공간

프로젝트에는 공예 유형, 작품 유형, 상태, 표지, 연결된 실과 사용량, 여러 카운터, 뜨개 노트, 대바늘·코바늘 도안과 기법 정보를 저장할 수 있습니다.

홈은 최근 업데이트한 작품을 우선 표시합니다. 작품을 열면 메인 카운터, 재료, 도안, 노트 및 기법으로 이동할 수 있습니다. 상태 변경은 목록 정리용이며 내용을 삭제하지 않습니다.

회원은 작품과 연결된 실, 카운터, 도안, PDF, 이미지를 하나의 `.lineopus` 파일로 내보낼 수 있습니다. 가져오기는 별도 사본을 만들며 같은 이름의 기존 작품을 덮어쓰지 않습니다.

<a id="counter"></a>
## 뜨개 카운터

- 단／라운드와 코를 함께 기록하거나 단일 값 카운터로 사용할 수 있습니다.
- 고정 코 수, 늘림·줄임 규칙, 자동 진행, 확인 또는 수동 완료를 설정합니다.
- 여러 카운터 탭을 추가하고 이름 변경, 정렬, 삭제할 수 있습니다.
- 현재 규칙을 뜨개 계획 작업으로 저장하고 다음 구간을 추가할 수 있습니다.
- 대바늘 도안, 읽기 전용 뜨개 노트, 텍스트 설명의 세 가지 표시 모드를 지원합니다.
- 작업 로그는 프로젝트 분석에 사용할 수 있습니다.

<a id="knitting-chart"></a>
## 대바늘 도안

격자 셀에 색상과 스티치 기호를 저장하여 대바늘 도안, 배색, 십자수, 픽셀 아트를 만듭니다.

- 행과 열, 셀 크기, 읽기 방향을 설정합니다.
- 시스템 또는 사용자 지정 기호를 선택합니다.
- 기호 그리기, 채우기, 지우기, 실행 취소와 다시 실행을 사용합니다.
- 영역 선택으로 복사, 잘라내기, 붙여넣기, 교체, 반복 배치를 수행합니다.
- 색상, 템플릿, 사용자 기호를 저장합니다.

이미지에서 만드는 방법은 두 가지입니다.

- **일반 이미지**: 사진이나 그림을 자르고 회전하며 비율과 위치를 조정한 뒤 픽셀 색상으로 변환합니다.
- **격자 도안**: 이미 셀 선이 있는 이미지나 PDF의 바깥 테두리와 가로·세로 간격을 보정한 후 캔버스 셀에 맞춥니다. 이후 픽셀 그리기 또는 기호 그리기를 선택합니다.

적용 전에 미리보기로 기존 캔버스가 예상치 않게 바뀌지 않는지 확인하세요. 도안을 프로젝트 카운터에 연결하면 현재 단을 따라갈 수 있습니다.

색상 그리기는 **자연색**, **간결한 색상 블록**, **흑백** 구성을 제공합니다. 일반 이미지는 빠른 미리보기를 사용하거나 **그리기 구성 확인**에서 설정과 색상 그룹을 수정할 수 있습니다. 격자 픽셀 도안은 보정된 셀별로 색상을 추출한 뒤 확인 화면을 엽니다. 색상 변경, 위치 확인, 병합, 분할, 무시가 가능하며 기호 그리기에서는 분류된 결과의 뜨개 기호와 셀 배경색을 수정할 수 있습니다. 모든 방식은 먼저 미리보기만 만들고 **캔버스에 적용**한 뒤에 작품에 반영됩니다.

저장 때문에 캔버스 조작이 중단되지 않도록 편집 중에는 정기적으로 자동 저장하지 않습니다. 앱을 백그라운드로 전환하거나 내보내기 화면으로 이동하거나 ‘닫기’를 누르면 자동 저장합니다. ‘닫기’는 저장이 끝난 뒤 이전 화면으로 돌아갑니다. 상단의 ‘저장’ 버튼으로 언제든 즉시 수동 저장할 수도 있습니다.

<a id="crochet-chart"></a>
## 코바늘 도안

코바늘 편집기는 자유 캔버스를 사용합니다.

- 다국어 시스템 기호와 사용자 지정 기호를 사용합니다.
- 기호를 이동, 회전, 확대·축소, 복제, 삭제하고 레이어 순서를 변경합니다.
- 여러 기호를 그룹화하고 함께 변형합니다.
- 직선이나 곡선 경로에 반복 기호를 배치하고 방향, 간격, 시작점을 조정합니다.
- 빠른 채우기, 기본 크기, 카운터 및 캔버스 가이드를 사용합니다.
- 작품을 저장하고 도안을 내보냅니다.

### 편집 도움말

- 기호 보관함은 이름, 약어, 키워드로 검색할 수 있으며 최근에 선택한 기호는 ‘사용됨’에 우선 표시됩니다.
- 작품 설정에서 이후 추가되는 기호의 선 두께와 기본 크기를 조절할 수 있습니다. 기둥코 2~6 기호는 곡률 핸들로 구부릴 수 있습니다.
- 레이어 관리에는 일반 그룹의 구성원이 재귀적으로 표시됩니다. 그룹을 펼치거나 접고, 요소 위치 찾기, 표시·숨기기, 잠금을 사용할 수 있습니다.
- 다중 선택 모드에서는 요소를 탭해 선택에 추가하거나 제거할 수 있고, 새로 영역을 지정하면 기존 선택이 교체됩니다. 선택된 요소 하나를 드래그하면 그룹을 만들지 않고 전체 선택을 함께 이동하며, 빈 캔버스를 탭하면 다중 선택을 종료합니다.
- ‘스마트 스냅’은 기본으로 켜져 있습니다. 드래그하는 동안 중심 십자선이 항상 요소를 따라가며, 요소 중심은 현재 보이는 캔버스 중심에 스냅되고 시각적 가장자리나 중심은 다른 표시 중인 요소에 정렬됩니다. 스냅되면 강조 안내선과 한 번의 가벼운 햅틱 피드백이 제공됩니다. ‘설정 > 편집 보조’에서 스냅을 꺼도 중심 십자선은 유지됩니다.
- 저장 때문에 캔버스 조작이 중단되지 않도록 편집 중에는 정기적으로 자동 저장하지 않습니다. 앱을 백그라운드로 전환하거나 내보내기 화면으로 이동하거나 ‘닫기’를 누르면 자동 저장합니다. ‘닫기’는 저장이 끝난 뒤 이전 화면으로 돌아갑니다. 상단의 ‘저장’ 버튼으로 언제든 즉시 수동 저장할 수도 있습니다.
- 내보내기에서 PNG/PDF, 해상도, 여백, 배경을 설정하고 제목, 메모, 기호 범례, 작성자와 저작권 정보를 선택하여 포함할 수 있습니다.

대바늘과 코바늘 도안은 서로 다른 데이터 형식입니다. 각 형식에 맞는 편집기에서 만들고 가져오세요.

<a id="knitting-notes"></a>
## 뜨개 노트

PDF 또는 이미지 도안을 가져오고 원본 PDF를 직접 변경하지 않은 채 편집 요소를 겹쳐 표시합니다.

- 카메라, 사진 또는 파일에서 노트를 만듭니다.
- 확대·축소, 페이지 이동, 썸네일 및 페이지 태그를 사용합니다.
- 이미지, 텍스트, 정보 표시, 자, 도안 링크, 격자 분할 프레임을 추가합니다.
- 펜, 지우개, 실행 취소·다시 실행 및 요소 관리를 사용합니다.
- 보조 카운터를 사용하고 노트를 프로젝트나 카운터에 연결합니다.

자세한 설명은 도움말 센터에서 “뜨개 노트”를 선택하세요.

<a id="other-tools"></a>
## 기타 도구

- **색상 도구**: 이미지에서 색상을 추출하고 팔레트와 색상 기록을 저장합니다.
- **기호 보관함**: 대바늘·코바늘 기호를 검색하고 개인 기호나 메모를 관리합니다.
- **신체 치수**: 의류 프로젝트에 필요한 치수를 기록하고 기본 코·단 환산을 미리 봅니다.
- **계산 도구**: 게이지, 길이, 무게, 크기 계산을 돕습니다.
- **달력과 알림**: 실과 작품 기록을 확인하고 재고 관련 알림을 설정합니다.

<a id="data-transfer"></a>
## 가져오기·내보내기·동기화

설정의 iCloud 업로드／다운로드는 전체 보관함 백업입니다. 현재 백업에는 데이터베이스와 필요한 PDF·이미지 파일이 포함됩니다. 다운로드는 로컬 데이터를 클라우드 백업으로 교체하며 항목별 병합이 아닙니다.

- `.lineopus`: 하나의 완전한 프로젝트와 연결된 자료.
- `.pxproj` 등: 편집 가능한 대바늘 도안.
- PDF／이미지: 뜨개 노트 원본 또는 공유 결과.
- CSV: 실 기록 일괄 전송.

<a id="membership"></a>
## 멤버십

기본 열람과 기존 콘텐츠 사용은 가능한 범위에서 유지됩니다. 일괄 전송, 완전한 프로젝트 패키지, 추가 생성 한도 및 일부 전문 편집 기능은 회원 기능입니다. 정확한 조건은 작업 시 표시되는 안내를 확인하세요. 기기 변경 후에는 설정의 회원 화면에서 구매 복원을 사용할 수 있습니다.

<a id="settings-help"></a>
## 설정과 도움말

설정에서 테마, 다섯 언어, 업데이트 기록, 피드백, 앱 기능 안내, 개인정보 처리방침과 이용 약관을 열 수 있습니다. 주요 편집기는 라이트 모드와 다크 모드를 지원합니다.

<a id="faq"></a>
## 자주 묻는 질문

### 가져온 뒤 사본이 생기는 이유는 무엇인가요?

기존 작업을 보호하기 위해 완전한 프로젝트와 도안은 기본적으로 독립 사본으로 가져옵니다.

### 다른 기기에서 PDF나 이미지가 보이지 않습니다

파일을 포함한 최신 백업을 사용하고 업로드／다운로드가 끝날 때까지 기다리세요. 이전 데이터베이스 파일만 복사하면 PDF와 이미지가 빠질 수 있습니다.

### 튜토리얼을 삭제한 뒤 다시 생기지 않는 이유는 무엇인가요?

튜토리얼은 처음 빈 보관함을 확인할 때 한 번만 설치됩니다. 삭제는 더 이상 필요하지 않다는 선택으로 처리됩니다.

### 문제를 어떻게 신고하나요?

“설정 → 피드백”에서 작업 순서, 예상 결과, 실제 결과를 작성하고 화면 문제라면 스크린샷도 첨부하세요.

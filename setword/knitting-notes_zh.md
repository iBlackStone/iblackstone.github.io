# 编织笔记使用帮助

编织笔记用于导入和阅读 PDF 图解，并在页面上添加尺子、文字、图片、提示、画笔轨迹和计数器。原始 PDF 不会因页面组件或画笔操作而被直接改写。

## 创建与打开项目

1. 进入工具页，选择“编织笔记”。
2. 选择要导入的 PDF 文件。
3. 导入完成后，项目会出现在编织笔记项目列表中。
4. 点击项目即可继续上次的阅读位置和编辑状态。

免费用户最多创建 1 个编织笔记项目。删除项目属于不可撤销操作，请确认不再需要后再删除。

## 编辑器界面

### 顶部工具栏

- **关闭**：返回项目列表并保存当前阅读状态。
- **撤销**：撤销最近一次支持的画笔或组件操作。
- **重做**：恢复刚刚撤销的操作。
- **更多**：重命名、保存或删除当前项目。

### 显示与隐藏工具栏

点击 PDF 空白区域可以收起顶部和底部工具栏，获得更大的阅读区域。工具栏收起后，可点击屏幕边缘的小按钮重新显示。

### 基本阅读手势

- 单指滑动：浏览 PDF。
- 双指捏合：缩放页面。
- 点击页面组件：选中组件并显示悬浮控制栏。
- 滑动或缩放 PDF：当前组件会失去焦点，方便继续阅读。

## 底部工具栏

| 工具 | 作用 |
|---|---|
| 页面 | 展开页面缩略图和页面标签，快速跳转 |
| 组件 | 选择图片、提示按钮、图解链接、文本或尺子 |
| 移动 | 返回 PDF 阅读、滑动和缩放模式 |
| 画笔 | 选择画笔并在当前 PDF 页面绘制 |
| 橡皮擦 | 擦除画笔绘制的轨迹 |
| 计数器 | 展开或收起项目计数器 |
| 更多 | 打开元素管理、默认颜色、画笔管理和尺子组件管理 |

## 页面与标签

点击底部“页面”按钮可以打开缩略图栏：

- 点击缩略图跳转到对应页面。
- 顶部标签列表可以直接跳转到已标记页面。
- 缩略图底部会显示该页标签。
- 点击选中缩略图上的 `...`，可以新增、编辑或删除页面标签。
- 每个页面最多设置 1 个标签。

免费用户最多创建 2 个页面标签，已有标签仍可编辑或删除。

## 添加页面组件

1. 点击底部“组件”按钮。
2. 选择需要的组件。
3. 组件按钮会显示为选中状态。
4. 点击 PDF 页面中的目标位置，组件将以触点为中心添加。
5. 添加完成后，编辑器会自动回到移动模式。

新增页面组件为会员功能。已有组件仍可查看和编辑。

### 支持的组件

#### 尺子

尺子可用于跟踪当前阅读行或列。默认提供横向尺子、纵向尺子和 45° 尺子，并可在尺子组件管理中调整预设。

#### 文本

文本组件支持多行内容。拉伸文本框后，可以显示更多文字。文本设置支持文字颜色、字号、对齐、复制和粘贴。

#### 图片

图片可以来自相机、文件或相册。图片按原始宽高比添加，只支持等比缩放和旋转，不支持四边独立拉伸。

#### 提示按钮

提示按钮在页面上显示为固定大小的 `i` 图标。点击图标可查看提示内容，双击弹层中的文字区域可开始编辑。

#### 图解链接

图解链接用于关联 App 内已有的棒针图解。关联后可以显示图解缩略图，并从 PDF 页面直接跳转到对应棒针图解。

## 编辑页面组件

点击组件后会出现虚线选中框、交互点和底部悬浮控制栏。根据组件类型，可以进行以下操作：

- 拖动组件内部：移动位置。
- 拖动边缘交互点：改变宽度或高度。
- 拖动右上角交互点：等比缩放。
- 拖动顶部交互点：旋转。
- 颜色：修改组件颜色。
- 透明度：调整组件背景透明度。
- 变换：快速设置角度或缩放比例。
- 删除：删除当前组件。

并非所有组件都支持全部操作。例如图片不能独立拉伸四边，提示按钮和图解链接不支持旋转或缩放。

### 锁定与解锁

- 锁定后，悬浮控制栏只显示“解锁”按钮。
- 锁定的尺子仍可整体拖动，但不能拉伸、缩放、旋转或修改样式。
- 锁定的文本、图片、提示按钮和图解链接等其他组件不能移动或编辑。
- 点击“解锁”后，完整控制栏和交互点会重新出现。

## 画笔与橡皮擦

点击“画笔”选择预设画笔后，即可在当前页面绘制。

- 画笔预设包含形状、粗细、颜色和透明度。
- 可以在画笔管理中新增、编辑、删除和排序画笔。
- 编辑器优先显示排序靠前的画笔。
- 橡皮擦只擦除画笔轨迹，不会删除尺子、文本、图片或其他组件。
- 页面组件请通过选中后的删除按钮，或元素管理进行删除。

新增画笔预设为会员功能，默认画笔和已有画笔仍可使用。

## 计数器

每个编织笔记项目至少包含 1 个计数器。

- 点击数字：计数加 1。
- 点击 `-`：计数减 1。
- 点击清零：将计数恢复为 0。
- 点击 `...` 或长按计数器：打开计数器菜单。
- 菜单支持编辑、快捷输入数字、改色、重命名和删除。
- “行数”是计数器最大值，超过最大值后会从 0 重新开始。
- 项目的第一个计数器不能删除。

每个横向页面最多展示 3 个计数器。超过 3 个时可左右滑动切换，顶部页码指示器会提示当前页。右上角按钮可以切换完整和紧凑高度。

免费用户最多使用 3 个计数器。

## 元素管理

通过底部“更多”进入“元素管理”，可以集中查看当前项目中的组件和画笔笔记。

- 左侧预览用于识别元素内容和大致轮廓。
- 点击元素可定位到对应页面和位置。
- 可以锁定或解锁组件。
- 可以删除不再需要的元素。
- 相邻的画笔轨迹会组合为较完整的笔记预览，便于识别。

## 默认颜色、画笔和尺子预设

底部“更多”提供以下通用设置：

- **默认颜色**：影响之后新增的页面组件。
- **画笔管理**：管理所有编织笔记项目共用的画笔预设。
- **尺子组件管理**：管理所有编织笔记项目共用的尺子预设。

修改通用预设不会自动改变已经添加到项目中的组件或轨迹。

## 保存、撤销与恢复

- 阅读页、页面标签、计数器和大部分编辑操作会随项目保存。
- 需要立即确认保存时，可使用顶部“更多 > 保存”。
- 撤销与重做适用于当前编辑会话中支持的组件和画笔操作。
- 退出项目再重新进入后，撤销栈可能不会保留，但已保存的页面内容会恢复。

## iCloud 与多设备

编织笔记的数据备份包含项目数据库、PDF 文件和 PDF 相关图片资产。启用并完成 App 的 iCloud 数据同步或恢复后，另一台设备才能看到完整项目。

请注意：

- iCloud 备份不是多人实时协作。
- 大型 PDF 的上传和恢复可能需要更多时间。
- 恢复完成前请不要删除原设备上的项目或 PDF 文件。
- 旧版本仅包含数据库的备份可能无法恢复 PDF 本体。

## 常见问题

### 为什么选中组件后 PDF 不能滑动？

组件命中区域的手势优先级高于 PDF。点击空白区域、切换到移动模式，或开始缩放 PDF，即可退出组件焦点。

### 为什么橡皮擦不能删除尺子或文本？

橡皮擦只处理画笔轨迹。其他组件需要使用组件控制栏中的删除按钮，或在元素管理中删除。

### 为什么锁定后的尺子还能移动？

尺子需要在阅读过程中持续调整位置，因此锁定只固定它的尺寸、角度和样式。其他组件锁定后会完全固定。

### 为什么新增组件后需要再点击页面？

组件采用“先选择、再落点”的方式，避免自动添加在页面中心后还要手动移动。

### 为什么有些功能会显示会员提示？

免费版保留基础导入、阅读和有限额度。新增页面组件、增加画笔预设等高级能力需要会员权限。

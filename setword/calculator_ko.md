## 개요

이것은 편성 전용으로 설계된 메모리 기능이 포함된 카운터로, 핵심 기능은 두 가지로 나뉩니다. 하나는 **외부 카운터**로, 주로 PDF 또는 이미지 형식의 도해에 대한 행 카운팅 기능을 제공합니다. 다른 하나는 내부 카운터로, 주로 앱의 도해 편집기로 제작된 도해에 대응하며 기능이 더 풍부하여 행과 바늘 수를 동시에 카운팅할 수 있습니다.

카운터를 작품 내에서 사용할 때, 작품에 사용된 실 정보를 입력하면 자동으로 사용량 및 비용 등의 정보를 계산할 수 있습니다.

### 문서 설명

본 문서는 카운터의 UI 레이아웃에 따라 위에서 아래로 기능의 용도를 설명합니다.

### 새로 추가 (탭 추가 가능)

사용자 정의 이름을 설정할 수 있는 탭을 새로 추가하는 데 사용됩니다. 하나의 편성 작품은 여러 개의 탭으로 나눌 수 있으며, 각 탭은 완전히 독립되어 있으며 페이지 하단의 제어 영역 버튼만 공유합니다. 탭 내에서 생성되는 행·바늘 수 카운트, 로그, 도해 등 모든 데이터는 서로 독립적입니다.

### 탭 영역 (탐색 바 아래, 모드 전환 버튼 위)

#### 편집 버튼
* 탭 바의 가장 오른쪽에 위치하며, 이름 변경 및 삭제 작업에 사용됩니다.
* [삭제]는 되돌릴 수 없습니다. 삭제 작업을 실행하면 탭의 모든 데이터가 제거되므로 신중하게操作해 주세요.

#### 도해 표시 영역
도해를 표시하는 영역으로, 주로 두 가지 표시 형식이 있습니다.
* 내부 도해 표시: 현재 앱 내의 도해 편집기로 제작된 도해를 내부 도해라고 합니다.
* 외부 도해 표시: 앨범 또는 파일에서 가져온 이미지 또는 PDF 파일.

##### 외부 도해 영역
> 기능 영역 버튼 → [외부 도해] 버튼을 선택하면 표시됩니다.

* 오른쪽 상단 [십자 버튼]: 클릭하면 도해 영역에 중앙 정렬된 십자 구분선이 표시되어 복잡한 도해의 위치 지정에 사용됩니다.
* 도해 왼쪽 행 식별자: 도해의 행 위치를 간단히 표시하며, 스크롤 및 클릭이 가능하고 하단의 빨간색 글자가 현재 행을 나타냅니다.

##### 내부 도해 영역
> 기능 영역 버튼 → [내부 도해] 버튼을 선택하면 표시됩니다.

본질적으로 내부 편집기의 캔버스이며, 다양한 제어 버튼이 없는 점을 제외하고는 외부 도해의 가상 행 위치 지정과 달리 내부 도해의 행·바늘 수 위치 지정은 모두 실제 위치 지정입니다.

### 카운터 표시 영역
행과 바늘 수의 카운트를 표시하는 데 사용됩니다.
* 안정 모드 카운팅 지원
* 가변 모드 카운팅 지원 (행 수가 변경됨에 따라 바늘 수가 변하는 모드)
* 목표 설정 지원: 카운트가 행 목표에 도달하면 카운터 화면이 빠르게 깜빡이고 약간 진동하여 특정 단계의 작업 완료를 알립니다.
* 초기 행 설정 지원: 기본적으로 0행부터 카운트를 시작하며, 다양한 행 카운팅 습관에 대응하기 위해 1행부터 시작하도록 설정할 수도 있습니다.

### 전환 탭
이름 그대로 선택한 탭의 값이 카운팅 시 변경되므로 자세한 설명은 생략합니다.

### 로그 영역
사용자의操作에 의해 발생하는 행·바늘 수 변경 로그입니다.
* [S]는 안정 모드를, [C]는 가변 모드를 나타냅니다.
* 오른쪽 [M] 버튼: 현재操作 로그에 메모를 추가할 수 있으며, 메모는 오류 추적에 사용할 수 있습니다.
* 로그 영역은 기본적으로 작게 표시되며, 필요에 따라 확대할 수도 있습니다.
* 로그에 메모가 있는 경우 해당 로그를 클릭하면 메모를 확인할 수 있습니다.
* [+]操作 기록 로그는 검은색으로, [-]操作 기록 로그는 빨간색으로 표시됩니다.

### 편성 모드 영역
* 안정 모드: 각 행의 바늘 수가 동일하며, 안정적인 주기로 편성하는 모드.
* 가변 모드: 각 행의 바늘 수가 단계적으로 변하며,相대적으로 바늘 수가 변하는 주기로 편성하는 모드.

#### 안정 모드
안정 모드를 선택한 후 [설정]을 클릭하면 설정 팝업이 나타나며, 두 가지 매개변수를 입력해야 합니다.
* 한 행의 바늘 수 상한: 즉 한 행에 필요한 바늘 수를 나타냅니다.
* 편성해야 할 행 수.

설정 후 카운터 표시 영역에 현재 정보가 표시됩니다. 예:
* 화면 상단: 한 행 바늘 수: 10, 행 목표: 20
* 화면 하단: 현재 선택된 모드 및 [+][-] 클릭 시触发되는 단위 변경
* 화면 오른쪽: 현재 작업 진행 상태. 참고로 100%를 초과할 수 있으며, 예를 들어 행 목표가 20행이어도 사용자가 30행까지 편성할 수 있습니다.
* 목표 달성 시 카운터 화면이 깜빡여 사용자에게 알리고, 터치 디바이스의 경우 기기가 약간 진동합니다.

#### 가변 모드
가변 모드를 선택한 후 [설정]을 클릭하면 설정 팝업이 나타납니다. 필요한 매개변수가 많아 별도의 페이지에서 사용자操作을 받습니다.

모드 설정기
* 설정 매개변수가 다소 복잡하므로 페이지 하단에 자세한 규칙 설명이 있습니다. 여기서는 반복 설명하지 않습니다.

설정 후 카운터 표시 영역에 현재 정보가 표시됩니다. 예:
* 화면 상단: 3행마다 1바늘 감소, 4회 반복
* 화면 하단: 현재 선택된 모드 및 현재 행의 반복 횟수 안내, 물론 단위도 표시됩니다.
* 화면 오른쪽: 현재 작업 진행 상태. 참고로 100%를 초과할 수 있으며, 예를 들어 행 목표가 20행이어도 사용자가 30행까지 편성할 수 있습니다.
* 바늘 수 변경이 필요한 경우 및 목표 달성 시 카운터 화면이 깜빡여 사용자에게 알리고, 터치 디바이스의 경우 기기가 약간 진동합니다.

### 기능 영역
오래된 사용자들은 알다시피 카운터의 기존 제어 버튼은 페이지各处에 흩어져 있었습니다. 하지만 카운터 기능이 점차 풍부해지면서 필요한 사용자 상호작용도 증가했으므로, 관련 제어 버튼을 모아 분류하는 것이 시급했고 그 결과가 현재 보시는 기능 영역 팝업입니다.

#### 첫 번째 행 버튼

##### 차트 분석
**차트 분석**: 카운트 로그를 집계·분석하여 작성된操作 차트로, 색채가 풍부한 차트를 좋아하는 사용자에게 편성 과정을 회상하며 만족감을 줄 수 있습니다.

카운터를 편성 작품 내에서 사용할 때 차트 분석 기능이 더 풍부해지며 다음 기능을 제공합니다:
* 편성 작품은 실 기록 섹션과 연동되어 사용한 실 선택, 소모 중량 추적, 기존 가격에 기반한 소모 실 비용 자동 계산이 가능합니다.
* 이를 통해 설정된 연동 관계는 실 섹션에도 동기화되어 반영됩니다.
* 이러한 데이터 포인트의 연동을 통해 사용자는 각 작품의 전체 과정을 확인할 수 있습니다.
* 앱은 또한 테이블 형식으로 편성 작품을 설명하는 회상 기능을 제공합니다.

> 작품 모듈은 개발자가 많은 시간과 노력을 기울인 섹션이며, 카운터는 작품 모듈에서 편성 과정 카운팅의 한 단계에 불과합니다. 따라서 작품 모듈 사용을 권장하며, 만족감을 얻을 수 있습니다.

##### 내부 도해
내부 도해 버튼을 선택하면 페이지 상단의 도해 표시 영역이 도해 작품으로 전환되며,此时 내부 도해 제어 버튼을 통해 표시를 제어할 수 있습니다.

* 작품 가져오기: 대상 도해 작품(예: 목표 타이하이 도해)을 선택합니다. 처음 작품을 가져오지 않은 경우 도해 표시 영역에 간편한 가져오기 버튼도 제공되며, 팝업되는 작품 목록에서 선택할 수 있습니다.
참고: 작품 목록 데이터는 도구 섹션의 작품 영역에서 가져옵니다.

* 스타일 설정: 도해 편집기로 진입하여 편집기가 제공하는 일련의 기능을 통해 표시 스타일을 제어할 수 있습니다. 자세한 내용은 편집기의 [카운트 모드] 설명을 참고하세요.

* 카운트 중앙 정렬: 먼저 행만 제어합니다. 기본적으로 선택·활성화되어 있으며, 활성화 시 상단 도해 영역에서 선택된 현재 행이 항상 도해 영역 중앙에 위치하며, 비활성화 후에는 행이 자동으로 중앙 정렬되지 않습니다.

* 열 +-: 수동으로 위치를 제어해야 하며, 현재는 바늘 수와 연동되어 있지 않습니다. 사용자가 이 기능을 필요로 하는 경우 피드백해 주세요.

##### 외부 도해
외부 도해 버튼을 선택하면 페이지 상단의 도해 표시 영역이 외부 도해로 전환되며,此时 연동 설정 버튼을 통해 연동 방식을 간단히 조정할 수 있습니다.

* 도해 가져오기: 시스템 앨범 앱 및 iCloud 파일에서 가져오기를 지원합니다. 참고로 일반적으로 PDF 파일의 선명도가 이미지 형식보다 약간 우수하지만, 앱은 이미지 형식에도 벡터 렌더링 방식을 적용하여 선명도를 향상시킵니다.

* 연동 설정: 외부 도해의 내용이 다양하므로 앱은 해당 내용을 분석하여 동적으로 가상 행 높이를 계산하지 않으며, 이 부분 기능은 사용자가 사용자 정의할 수 있습니다. 연동 설정은 기본적으로 활성화되어 있으며,具体적인 사용 규칙은 팝업 내에 자세히 설명되어 있으므로 여기서는 반복 설명하지 않습니다.

#### 제어 영역 버튼
* 로그 크기: 로그 영역의 표시 크기를 제어하며, 기본적으로 작게 표시됩니다. 자세한 로그를 회상해야 할 경우 확대하여 확인할 수 있습니다.
* 로그 지우기: 현재 탭의 모든 로그를 지우며,操作은 되돌릴 수 없습니다. 로그에 연결된 메모 등의 정보도 함께 지워집니다. 참고로 차트 분석 데이터는 로그 데이터에서 가져오므로 해당 차트도 초기화됩니다.
* 통계 지우기: 선택된 행 수 또는 바늘 수를 간편하게 초기화하는 데 사용됩니다. 모두 지우려면 두 번에 나누어 지우기操作을 수행해야 하며, 관련 초기화操作에는 이차 확인이 있어 사용자 데이터를 보호합니다.

### 하단 버튼 영역
* [-] : 감산操作. 행 또는 바늘 수를 감산합니다.
* [+] : 가산操作. 행 또는 바늘 수를 가산합니다.
* 중앙 단위 영역: 한 번의 가감산操作 단위. 참고로 단위 컨트롤을 빠르게 스크롤하고 스크롤 애니메이션이 끝나지 않은 상태에서 [+][-] 버튼을 클릭하면 새 단위가 적용되지 않습니다. 기준은 중앙 카운터 화면을 참고하세요. 언제든지 [+][-]를 클릭할 때 프로그램이 계산에 사용하는 단위는 카운터 화면 하단에 표시된 단위 숫자입니다.

### 연동 섹션
앱의 현재 기존 섹션은 어느 것이나 독립적으로 존재하는 것이 아니라 편성 관심을 중심으로 긴밀하게 연동되어 있습니다. 현재 카운터는 편성 작품의 하위 분기로 속하며, 편성 카운팅을 보조하는 데 사용됩니다.

> 카운터의 일부 기능은 영구 구매 사용자 또는 구독 사용자에게만 제공됩니다. 이러한 복잡하고 지속적으로 업데이트되는 앱을 유지하려면 개발자가 끊임없이 시간과 노력을 투입해야 합니다.

1. **전문 용어 통일**:
   - "编织计数器" → "편성 카운터" (한국 편성 앱에서 일반적인 호칭)
   - "图解" → "도해" (편성 관련 문서에서 표준 용어)
   - "行/针" → "행/바늘 수" (편성 분야 표준 표현)
   - "稳定模式/变化模式" → "안정 모드/가변 모드" (기능 특성을 명확히 반영)
   - "选项卡" → "탭" (UI 영역의 국제적 공통어)

2. **기술적 정확성**:
   - "记忆性计数器" → "메모리 기능이 포함된 카운터" (기능 핵심 정확 전달)
   - "矢量渲染" → "벡터 렌더링" (기술 용어 표준 번역)
   - "不可逆" → "되돌릴 수 없음" (사용자에게 경고하는 명확한 표현)
   - "二次确认" → "이차 확인" (앱操作 표준 표현)

## 参考価値

### データ参考説明
掲載データはあくまで参考です。編み手の張り具合や編み模様の違いにより、同じ糸と針サイズを使用しても、編み上がるスワッチの密度は異なる場合があります。そのため手作り編みでは、通常、まずスワッチを編み上げ、水洗い・定型処理を行った後、実際の密度を測定し、最終作品の針数と段数を決定することで、完成品のサイズズレを回避します。

### 一、コア目標
プログラミングの観点から「グリッドベースの棒針/鉤針スワッチシミュレーター」を実現することを目指し、**糸の物理パラメータをグリッドの視覚化パラメータにマッピングし、1針を1つのグリッドセルに対応させる**ことを核心とします。

#### コア技術フロー
```
入力パラメータ（線径/番手/撚り数等）→ 標準編み密度計算（10cmあたり針数/段数）→ 10cmスワッチのグリッド行列数換算 → 棒針/鉤針工法のグリッドスタイル適応 → App端でグリッド描画
```

### 二、基本データ基準
すべての計算式は以下の国際/業界標準に基づいており、データの正確性を保証します。

| 標準番号                | 標準名                              | コア適用シナリオ                     |
| :---------------------- | :---------------------------------- | :----------------------------------- |
| ISO 2314:2010           | 繊維製品 - 糸の番手測定方法          | 番手と線径、長さの換算               |
| ASTM D2253-19           | ニット生地の単位面積重量及び密度試験方法 | 編み密度（10cmあたり針数/段数）の計算基準 |
| JIS L1096:2010          | 織物及びニット生地の試験方法          | 棒針/鉤針密度の業界参考範囲           |
| 国際羊毛局（IWTO）標準  | 羊毛糸の線径と撚り数の対応関係        | 撚り数による実際線径の補正           |

### 三、編み密度計算（ASTM D2253標準）
編み密度（10cmあたり針数）はグリッドマッピングの核心であり、「実際の線径＋工法種類」を組み合わせて計算し、JIS L1096の密度範囲を参考とします。

| 実際線径（mm） | 棒針密度（10cmあたり針数） | 鉤針密度（10cmあたり針数） | 棒針段数/10cm | 鉤針段数/10cm |
| :------------- | :-------------------------- | :-------------------------- | :------------ | :------------ |
| 0.2~0.3        | 27~32（細糸）                | 25~30                       | 23~26         | 18~21         |
| 0.3~0.5        | 21~26（中細糸）              | 20~25                       | 19~22         | 15~18         |
| 0.5~0.8        | 16~20（中太糸）              | 15~20                       | 15~18         | 12~15         |
| 0.8~1.2        | 12~15（太糸）                | 10~15                       | 11~14         | 8~11          |
| >1.2           | 7~11（超太糸）               | 7~10                        | 7~10          | 5~8           |

**例**：実際線径0.25mm（2撚り細糸）→ 棒針密度=29針/10cm → 棒針段数=29×0.8=23.2段/10cm

#### 1. コア定義
編み密度 = 規定寸法（10cm）内の **横方向針数（Width Gauge）** / **縦方向段数（Height Gauge）**  
- 横方向針数：10cm幅内の編み針数（作品の幅サイズを決定）；  
- 縦方向段数：10cm高さ内の編み段数（作品の長さサイズを決定）。

#### 2. 標準試験手順
ASTM D2253では「スワッチ編み＋標準化測定」による正確性の保証を要求し、手順は以下の通りです：
1. **スワッチ編み**：  
   - 実際のプロジェクトで使用する糸と針を用い、少なくとも15cm×15cmのスワッチを編む（端の変形による測定誤差を回避）；  
   - 編み模様は完成品と一致させる（平編み、ブリオッシュ編み等、編み目の種類により密度差が大きくなる）。
2. **スチーム定型（重要工程）**：  
   - 糸の素材（羊毛/コットン/化学繊維）に応じてスチームアイロンで処理（蒸気を当てるのみ、押さえたり引っ張ったりしない）；  
   - 自然乾燥後に測定（定型前のスワッチは収縮する可能性があり、密度計算の誤差につながる）。
3. **精密測定**：  
   - 横方向：スワッチの中心領域（端から2-3cm避け）で10cmを測定し、含まれる針数を数える（1針に満たない場合は小数で表記、例：18.5針）；  
   - 縦方向：同じ中心領域で10cmを測定し、含まれる段数を数える；  
   - 2-3回測定を繰り返し、平均値を採用（正確性向上）。
4. **計算結果**：  
   例：10cm内に横方向18針、縦方向24段 → 編み密度は「18針 × 24段 / 10cm」。

### 四、棒針と鉤針
「短/中/長」で二級編み目分類を統一的に識別

| 汎用編み目グレード | 棒針対応関係       | 鉤針対応関係       | コア密度特徴       | 補正係数 |
|:--------------------|:--------------------|:--------------------|:--------------------|:----------|
| 短（Short）         | 平編み（Stockinette）| 短針（SC）          | 最も緻密（基準）   | 1.0       |
| 中（Medium）        | 逆平編み（Reverse） | 中長針（HDC）       | 中程度のルーズ     | 0.85      |
| 長（Long）          | ブリオッシュ編み（Brioche）| 長針（DC）      | 非常にルーズ       | 0.7       |
